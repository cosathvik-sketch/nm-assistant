Set Up Cloud Sync

Connect a free Supabase database so your library syncs across all devices automatically.

1Go to supabase.com → New Project (free)
2Click SQL Editor → paste and run this:
create table nm_library (id bigint primary key, tab text, marker int, label text, input text, output text, saved_at text, created_at timestamptz default now());
3Go to Settings → API → copy URL and anon key below
Supabase free tier is permanent and has no expiry. Your data is private to your project.
